# 🌊 Waveian - The Invisible Sound Experience

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![Platform: Windows](https://img.shields.io/badge/Platform-Windows-0078d7.svg)](https://www.microsoft.com/windows)

**Waveian** is a premium, seamless background music player for Windows 10/11. Built with Electron and inspired by the Fluent Design System, it lives invisibly in your system, providing high-quality audio through a powerful "Spotlight" search interface.

---

## ✨ Features

- **🔍 Spotlight Search:** Press `Ctrl + Shift + Space` to find and play any song instantly.
- **🤖 Smart Media Detection:** Automatically pauses when you start watching a video or join a call, and resumes when you're done.
- **💎 Glassmorphism UI:** A beautiful, translucent interface that feels like a native part of Windows 11.
- **🚀 Global Shortcuts:**
  - `Ctrl + Shift + 1/2`: Volume Down/Up.
  - Media Keys: Support for Play/Pause, Next, and Previous from anywhere.
- **🛡️ Secure Google Login:** Custom login flow that bypasses common "Insecure Browser" restrictions.
- **🔔 Discreet Notifications:** Bottom-left track notifications keep you informed without interrupting your work.

## 🛠️ Installation

### For Users
1. Download the latest `.exe` from the [release](release/) folder.
2. Run the installer.
3. Use the search bar to find your first song!

### For Developers
1. Clone the repository:
   ```bash
   git clone https://github.com/Rarafaza5/Waveian.git
   ```
2. Navigate to the source folder:
   ```bash
   cd OpenSource
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the app:
   ```bash
   npm start
   ```
